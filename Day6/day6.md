# Day 6 - AI-Powered Resume Optimization

## Objective

Learn how AI can improve resume quality and ATS (Applicant Tracking System) compatibility using Claude AI.

---

* Read the provided resources
* Opened a new conversation in Claude

* Created an optimized one-page resume
* Saved the optimized version

* Committed and pushed changes to GitHub

## Tools Used

* Claude AI
* ATS Resume Analysis
* GitHub
* Markdown

---

## Process

### Step 1: Resume Upload

Uploaded the existing resume to Claude AI.

### Step 2: ATS Analysis

Claude analyzed the resume and identified:

* Missing keywords
* Formatting issues
* Content improvement opportunities
* ATS compatibility concerns

### Step 3: Resume Optimization

Generated an ATS-friendly one-page resume with:

* Improved formatting
* Better keyword optimization
* Clearer project descriptions
* Enhanced readability

### Step 4: Documentation

Documented the entire workflow, screenshots, and learnings in GitHub.

---

## Key Learnings

* ATS systems play an important role in resume screening.
* Relevant keywords improve resume visibility.
* AI can quickly identify weaknesses in a resume.
* One-page resumes are often more effective for freshers.
* Proper formatting improves both ATS and recruiter readability.

---

## Results

### Original Resume

* Basic formatting
* Limited keyword optimization
* Lower ATS compatibility

### Optimized Resume

* ATS-friendly structure
* Improved keyword matching
* Better readability
* Professional presentation
