# Day 21 - Vehicle Fuel Analytics Dashboard

## Objective

Build a data-driven Vehicle Fuel Analytics Dashboard that analyzes fuel efficiency, operating costs, maintenance expenses, CO₂ emissions, and E85 viability using CSV data.

---

## Focus Area

**Data Analytics & Interactive Dashboard Development**

The goal was to transform raw vehicle fuel data into actionable insights through an interactive dashboard built with pure HTML, CSS, JavaScript, and SVG visualizations.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
* SVG Charts
* Data Analysis
* CSV Processing

---

## Problem Statement

Analyze vehicle fuel data and generate a dashboard that provides:

* Cost per kilometer analysis
* CO₂ emission comparison
* Maintenance cost analysis
* Refuel/recharge time comparison
* Vehicle age impact analysis
* E85 fuel viability assessment
* Personalized recommendations

---

## Metrics Computed

### Cost Metrics

* Average Cost per Kilometer
* Monthly Fuel Cost
* Fuel Comparison Analysis

### Environmental Metrics

* Average CO₂ Emissions per Kilometer
* Fuel-wise Emission Comparison

### Maintenance Metrics

* Maintenance Cost per Kilometer
* Age-wise Maintenance Trends

### E85 Analysis

* Pump Savings
* Running Cost Penalty
* Break-even Fuel Price
* E85 Adoption Score

---

## Dashboard Features

### KPI Cards

* Current Fuel Cost/km
* E85 Cost/km
* Premium Comparison
* Break-even Price
* Monthly Running Cost

### Visualizations

* Fuel Cost Comparison Bar Chart
* CO₂ Emission Doughnut Chart
* Cost vs Vehicle Age Line Chart
* E85 Score Gauge

### Fuel Recommendation Cards

Each fuel type includes:

* Advantages
* Disadvantages
* Best Use Cases

---

## Dashboard Workflow

CSV Data
↓
Data Processing
↓
Metric Calculation
↓
SVG Chart Generation
↓
Dashboard Rendering
↓
Insights & Recommendations

---

## Key Learnings

* Data visualization improves decision-making.
* SVG provides lightweight and responsive charts.
* Fuel economics depend on multiple variables beyond pump prices.
* Vehicle age significantly impacts maintenance costs.
* Interactive dashboards make complex datasets easier to understand.

---

## Challenges Faced

* Processing fuel-specific metrics
* Building charts without external libraries
* Designing responsive visualizations
* Implementing dynamic KPI calculations

---

## Outcome

Successfully developed a responsive fuel analytics dashboard that converts raw CSV data into actionable cost, efficiency, and environmental insights.

---

## Conclusion

This project demonstrated how data analytics and frontend engineering can be combined to build decision-support tools that help users compare fuel options and optimize vehicle operating costs.
