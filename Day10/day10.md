# Day 10 - AI Portfolio Website Generation

## Objective

Build a personal portfolio website using Claude AI by providing personal information, projects, skills, and experience, then generate a complete HTML portfolio that can be viewed locally or deployed online.

---

## Problem Statement

Create a professional portfolio website using AI-generated code and showcase your:

* Profile Information
* Skills
* Projects
* Education
* Experience
* Contact Information

The final output should be a fully functional HTML portfolio website.
