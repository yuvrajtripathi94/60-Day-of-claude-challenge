
# Day 28 – Hospital Admission Readiness Simulator

## Objective

Build an interactive Hospital Admission Readiness Simulator that evaluates whether a patient is ready for hospital admission by analyzing clinical, insurance, documentation, bed availability, and prior authorization factors.

---

## Focus Area

**Healthcare Workflow Simulation & Admission Readiness**

The objective of this project was to simulate the real-world hospital admission process, identify potential risks before admission, and improve admission readiness through interactive workflow actions.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Claude AI

---

## Project Overview

Hospital admissions involve multiple checkpoints before a patient can be admitted successfully. This simulator models the complete admission workflow by evaluating documentation, insurance, prior authorization, physician approval, and hospital resource availability.

Users can interact with different scenarios, resolve pending issues, and improve the overall admission readiness score.
