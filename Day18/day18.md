# Day 18 - Brain Dump Action Planner Custom Skill

## Objective

Create a reusable Claude Custom Skill that transforms unstructured notes into actionable insights, summaries, risks, blockers, and prioritized action items.

---

## Focus Area

**AI-Powered Productivity & Knowledge Management**

The goal was to build a reusable workflow that can process different types of notes and automatically generate structured outputs without repeatedly providing detailed instructions.

---

## Tools Used

* Claude AI
* Claude Custom Skills
* Interactive Dashboards
* Prompt Engineering

---

## Problem Statement

People often collect information in unstructured formats such as:

* Meeting Notes
* Brainstorming Sessions
* Class Notes
* Voice Memo Transcripts
* Project Discussions

These notes can be difficult to review and convert into actionable tasks.

The Brain Dump Action Planner skill solves this problem by automatically organizing information into structured outputs.

---

## Custom Skill Created

### Skill Name

**brain-dump-action-planner**

### Purpose

Convert raw notes into:

* Executive Summary
* Key Insights
* Action Items
* Risks
* Blockers
* Open Questions
* Priority Recommendations

---

## Workflow

```text
Raw Notes
    ↓
Claude Custom Skill
    ↓
Information Extraction
    ↓
Task Prioritization
    ↓
Dashboard Generation
    ↓
Actionable Insights
```
